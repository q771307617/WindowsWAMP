Description
This package allows you to run your website directly from CD/DVD, USB Stick or Hard Disk without installation or configuration as if it's a normal Windows application. No manually starting and stopping Apache, MySQL servers or services. Just double click the icon when you want to start it up, and close the window when you're finished working. It takes care of all the housekeeping for you!

Usage
Step 1   Place your website files on the subfolder called www.
Step 2   Place your MySQL database files on the subfolder called data.
Step 3   Run #appname and its window will show the respective webpage.
Notes
Note 1   #appname can run directly from CD, USB Stick or Hard Disk without installation.
Note 2   Default server port is 80 (automatically change if 80 is in use).
Note 3   Default MySQL username is root and the password is empty.
Note 4   Click FAVORITES button to open menu, click PHPMYADMIN to use phpMyAdmin.
